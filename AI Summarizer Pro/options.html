Getting Started

Follow these steps to set up your API key:

  • Visit Google AI Studio and sign in with your Google account
  • Click "Get API Key" in the left panel
  • Copy the generated API key
  • Paste it in the field above and click "Save Configuration"
  • Use "Test Connection" to verify everything is working

Note: Your API key is stored locally and never shared with third parties.
